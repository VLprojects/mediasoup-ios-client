# mediasoup-ios-client

Objective-C wrapper library for libmediasoupclient for building mediasoup iOS based applications.

# ⚠️ This project has been archived

This repository is no longer maintained.

Development has moved to **[mediasoup-client-swift](https://github.com/VLprojects/mediasoup-client-swift)**, a complete reimplementation with a Swift-first design, improved architecture, better reliability, and ongoing maintenance.

If you're starting a new project — or migrating an existing one — please use **mediasoup-client-swift** instead.
